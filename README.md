# dng

Wrapper around Adobe DNG Converter to use with *nix via wine

## Prerequisites

1. rust
2. wine
3. innoextract
4. make

## Installation

[![Packaging status](https://repology.org/badge/vertical-allrepos/dng.svg)](https://repology.org/project/dng/versions)

```shell
make
make install
```
